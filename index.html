About the App

FitTrack is a lightweight, browser-based fitness tracker built to help you log workouts, monitor calories, set goals, and stay consistent — all without needing any account or backend.

Smart Calorie Calc

Using the MET (Metabolic Equivalent) formula, FitTrack auto-calculates calories burned based on your exercise type, duration, body weight, and intensity level.

Tech Stack

Built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies — just clean, fast code that runs in any browser.

Features

Auto calorie calculation, workout logging, weekly activity chart, goal setting with progress bars, visitor counter, local data persistence, and a clean minimal UI.

A Note from the Team

FitTrack was built to make fitness tracking simple and accessible for everyone. Staying consistent is hard — this app is meant to make it just a little bit easier. Whether you're hitting the gym every day or just trying to stay active, FitTrack keeps you accountable. Built with ❤️ at Chitkara University by Lakshay Sharma & Keshav.
